# Smart Contract Honeypot Detector

A blockchain security tool that detects malicious honeypot smart contracts using static analysis and AST-based inspection.

## Features

- Solidity AST Parsing
- Smart Contract Static Analysis
- Honeypot Detection Logic
- Risk Scoring Engine
- Transfer Restriction Detection
- Owner Privilege Analysis
- Sell Lock Detection

## Tech Stack

- Node.js
- Express.js
- Solidity
- React
- JavaScript
- Web3 Security Concepts
